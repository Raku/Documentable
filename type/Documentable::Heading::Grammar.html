class Documentable::Heading::Grammar

Grammar to obtain metadata from headings

Table of Contents
1Type Graph
grammar Documentable::Heading::Grammar

Used to extract the necessary metadata from headings elements. It extracts four different values from a heading:

  • name.

  • subkind: which is classified in three different categories:

    • operator:

      • infix

      • prefix

      • postfix

      • circumfix

      • postcircumfix

      • listop

    • routine:

      • sub

      • method

      • term

      • routine

      • submethod

      • trait

    • syntax:

      • twigil

      • constant

      • variable

      • quote

      • declarator

  • category: operator or one of element of routine or syntax.

  • kind: Kind::Syntax or Kind::Routine. See Kind.
